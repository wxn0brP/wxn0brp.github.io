Supporting Libraries & Tools

A collection of lightweight tools to enhance your development experience.

  • Suglite: Watchdog for your processes - lightweight and scriptable
  • FalconFrame: Modular TS web framework for rapid development (server)
  • FlankerUi: TS UI framework (client)
  • Bumr: Easy bump npm packages versions tool
  • GlovesLink: Simple WebSocket library for efficient real-time apps
  • Gate Warden: Secure and customizable role-based access control system
  • AnotherCache: TS cache library
  • Pressure: Static website bundler
